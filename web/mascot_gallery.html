Mascotte · galerie d'animations

Chaque état de la mascotte joue en live. Les boucles tournent en continu ; les one-shots ont un bouton « rejouer ». Source unique : ajouter une anim dans js/mascot_states.js + ses keyframes dans styles/mascot.css la fait apparaître ici automatiquement.

Taille
Vitesse 1.0×

Pour ajouter une animation : une ligne dans web/js/mascot_states.js, ses @keyframes + une règle .mascot.is-<id> dans web/styles/mascot.css.
Servir via make run puis ouvrir http://localhost:8000/mascot_gallery.html.
